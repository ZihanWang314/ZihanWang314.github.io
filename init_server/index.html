Init Server

    how to make a new server work for AI practitioners instantly:

    here assume that your server is ubuntu 20.04:

    1. download nvidia driver
     sudo apt install nvidia-driver-515 nvidia-dkms-515

     if you did sonething wrong with the above step:
     1.1. remove incorrect nvidia drivers
      sudo apt-get purge nvidia-*
      sudo apt-get update
      sudo apt-get autoremove
     1.2 search latest nvidia driver
      apt search nvidia-driver

    2. reboot system
     sudo reboot

    3. download python
     sudo apt-get update
     sudo apt-get install python3
     with anaconda:
      wget https://repo.anaconda.com/archive/Anaconda3-2022.05-Linux-x86_64.sh # check the archive for more versions!
      bash Anaconda3-2022.05-Linux-x86_64.sh
      cd ~
      source .bashrc
      export PATH="~/anaconda3/bin:$PATH" # change the path to your local path
      conda init bash # change "bash" to your shell name

    4. download vscode extensions like pylance

    5. download pip
     sudo add-apt-repository universe
     sudo apt-get update
     sudo apt install python3-pip

    6. download requirements
     pip install torch torchvision torchaudio --extra-index-url https://download.pytorch.org/whl/cu116
     pip install numpy pandas transformers beautifulsoup4 flask jieba matplotlib scipy sentencepiece sklearn tqdm

    7. download cuda toolkit
     wget https://developer.download.nvidia.com/compute/cuda/repos/ubuntu2004/x86_64/cuda-keyring_1.0-1_all.deb # attention on the ubuntu version!
     sudo dpkg -i cuda-keyring_1.0-1_all.deb
     sudo apt-get update
     sudo apt-get -y install cuda

    8. download svn (for git hub clone):
     sudo apt-get install subversion

    Congrats that you have finished!
